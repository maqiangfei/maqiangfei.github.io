学习笔记 - JWT 使用

Json Web Token，通过数字签名的方式，以JSON对象为载体，在不同的服务终端之间安全的传输信息

作用

JWT最常见的场景是授权认证，一旦用户登录，后续每个请求都将包含JWT，系统在每次处理用户请求之前，都要先进行JWT安全校验，通过之后再进行处理

组成

JWT由3部分组成，用“.”拼接

这3部分分别为：

  • Header

  • Payload

  • Signature
