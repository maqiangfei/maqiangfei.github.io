Linux 配置静态IP

最小安装的Ubuntu 20.04及最小安装的CentOS 7可参考本文配置静态IP

CentOS 7

  1. 查看当前ip地址及网关地址

    输入命令ip addr查看ip地址

    输入命令ip route show查看网关地址

  2. 配置ifcfg-ens33

    修改BOOTPROTO=static、DNBOOT=yes、IPADDR=x.x.x.x、NETMASK=255.255.255.0、GATEWAY=x.x.x.x、DNS1=223.5.5.5

  3. 重启网络

Ubuntu 20.04

  1. 查看当前ip地址及网关地址

    输入命令ip addr查看ip地址

    输入命令ip route show查看网关地址

  2. 切换到/etc/netplan目录下

    注意：配置文件的名称可能有差异。并且它是一个yaml文件，注意语法缩进

  3. 使用cat命令查看00-install-config.yaml的内容

    从上面的输出可以看出网络接口为ens33，它开启了DHCP

  4. 配置静态IP代替DHCP，使用vi编辑配置文件

  5. 使用netplan命令使配置生效
